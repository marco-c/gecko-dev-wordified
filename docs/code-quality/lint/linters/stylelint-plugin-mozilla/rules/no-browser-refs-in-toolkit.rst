This
rule
prevents
toolkit
code
from
importing
browser
-
specific
CSS
or
SVG
files
.
It
is
applied
to
both
import
statements
and
CSS
property
values
using
the
url
function
.
This
ensures
that
toolkit
code
remains
independent
and
can
be
used
in
other
applications
/
contexts
where
browser
code
is
not
necessarily
available
.
Browser
code
can
depend
on
toolkit
code
but
toolkit
code
should
never
depend
on
browser
code
since
toolkit
code
can
be
used
in
other
applications
/
contexts
where
browser
code
may
not
be
available
.
See
additional
documentation
for
more
information
on
internal
URLs
<
/

/
