This
chapter
describes
the
NSPR
API
for
managing
linked
lists
.
The
API
is
a
set
of
macros
for
initializing
a
circular
(
doubly
linked
)
list
inserting
and
removing
elements
from
the
list
.
The
macros
are
not
thread
safe
.
The
caller
must
provide
for
mutually
-
exclusive
access
to
the
list
and
for
the
nodes
being
added
and
removed
from
the
list
.
